# Reliance Industries LSTM Stock Predictor

A full-stack deep learning application for predicting Reliance Industries (RELIANCE.NS) stock prices using a two-layer LSTM neural network.

## Results
| Metric | Value |
|--------|-------|
| R² Score | 0.9519 |
| RMSE | 22.86 INR |
| MAE | 17.18 INR |
| MAPE | 1.27% |

## Tech Stack
- **Model**: TensorFlow / Keras — 2-layer LSTM
- **Backend**: Flask REST API
- **Frontend**: Vanilla JS + Chart.js (Bloomberg-style dark dashboard)
- **Data**: yfinance (NSE — Mar 2021 to Feb 2025)

## Project Structure
```
reliance-lstm/
├── backend/
│   ├── app.py               ← Flask API
│   ├── preprocess.py        ← Data download + cleaning
│   ├── requirements.txt
│   └── data/                ← Generated by preprocess.py (gitignored)
└── frontend/
    ├── index.html
    ├── styles.css
    └── main.js
```

## Setup

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate      # Windows
pip install -r requirements.txt
python preprocess.py       # downloads and cleans data
python app.py              # trains model and starts API
```

### Frontend
```bash
cd frontend
python -m http.server 5500
```
Open http://localhost:5500 in your browser.

## Docker
```bash
docker-compose up --build
```
